Configuration Firebase requise
① Créez un projet Firebaseconsole.firebase.google.com → Nouveau projet
② Activez Firestore + AuthBuild → Firestore Database → Créer (mode production)
Build → Authentication → Email/Password → Activer
③ Règles Firestore
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /families/{familyId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == familyId; } } }
④ Collez votre config dans le HTMLProject Settings → App Web → SDK → remplacez le bloc firebaseConfig
